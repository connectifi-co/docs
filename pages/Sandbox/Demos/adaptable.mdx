# Adaptable Grid
[AdapTable](https://www.adaptabletools.com/) provides all the cutting-edge data-related functionality that advanced financial grid users require and expect out of the box.

## Application Name
In the **Application Info** step, set *Application Name* as `adaptableGrid`.

![Application Info Settings](../../../public/demos/imports/adaptable/info.png)

## Start URL & Intents

In the **Agent SDK** step, select SDK Type of *Web App*.

Set *Start URL* as (substituting the code for your directory as described [earlier](/Sandbox/Imports)): `https://demos.connectifi.app/platform/{YOUR_DIRECTORY_CODE}/adaptableGrid`

### Configure Supported Intents
This registers the intents the app supports with the directory, so that the app is fully discoverable and can be launched using the *start url* provided.

Select the Intent: `SendToWatchlist`

Select the following supported Contexts: 
- `fdc3.instrument`
- `fdc3.instrumentList` 
- `fdc3.portfolio`

**Note:** once youâ€™ve added an intent, click on the chip for the intent to get the picker list for contexts.

![Agent SDK Settings](../../../public/demos/imports/adaptable/agent.png)

## Finish

Got to the **Preview** step, check your settings, and select *Finish* to save the application.  

Test the app using its Start URL, it should load and connect to the directory. If it has registered intents, it will now be discoverable.
